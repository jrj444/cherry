# Switch 开关

表示两种相互对立的状态间的切换

# 基础用法

绑定 v-model 到一个 Boolean 类型的变量

<ch-switch v-model="value"></ch-switch>

<script>
export default {
  data() {
    return {
      value: true
    };
  }
}
</script>

# 禁用状态

设置 disabled 属性为 true即可禁用

<ch-switch v-model="value1" disabled></ch-switch>
<ch-switch v-model="value2" disabled></ch-switch>

<script>
export default {
  data() {
    return {
      value1: true,
      value2: false
    };
  }
}
</script>

# 属性

属性 说明 类型 可选值 默认值
value / v-model 绑定值 Boolean —— ——
disabled 是否禁用 Boolean —— false
