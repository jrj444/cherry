# Icon 图标

提供了一套常用的图标集合。

### 使用方法

通过组件的 ```name``` 属性来使用即可。例如：

<ClientOnly><IconDemo></IconDemo></ClientOnly>

```vue
  <ch-icon name="setting"></ch-icon>
  <ch-icon name="share"></ch-icon>
  <ch-icon name="edit"></ch-icon>
  <ch-icon name="github"></ch-icon>
```
